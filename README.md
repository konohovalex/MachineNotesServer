# MachineNotesServer

Rust web-server for https://github.com/konohovalex/MachineNotesAndroid

**Stack:**  
Actix-web  
Argon2  
Diesel  
Serde  
Jsonwebtoken

Runs on localhost.  
Requires installed PostgreSQL.  
Needs a .env file in root directory with PostgreSQL DATABASE_URL and JWT_SECRET being set.
